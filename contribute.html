Contribute to Protest With Your Wallet

Want to add more companies to protest? You can contribute directly by committing new pages to our GitHub repository.

How to Contribute

This project is open-source. Anyone can add new company protest pages, suggest edits, or improve existing guides. By contributing, you help others make informed choices and amplify consumer power.

To contribute:

  1. Visit the GitHub repository linked below.
  2. Fork the repository to your own GitHub account if you plan to submit changes back (recommended for pull requests).
  3. Add or edit HTML pages for companies you want to protest.
  4. If you forked: commit your changes and open a Pull Request to share them with the community.
Go to GitHub Repository

Why Contribute?

Collective action is most effective when shared. By documenting reasons to avoid certain companies and offering alternatives, you empower others to make values-driven decisions with their money. Your contributions help build a community of conscious consumers.
